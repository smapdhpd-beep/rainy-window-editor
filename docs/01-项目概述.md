# 项目概述

## 项目名称
rainy-window-editor — 雨夜车窗氛围生成器

## 灵感来源
参考 vibe coding 雨天编辑器：用户上传背景，通过程序化 Shader 叠加玻璃雨滴、冷凝雾气与文字排版，生成情绪化氛围图片/视频。

## 核心体验
- 雨滴在玻璃上自然滑落，对背后街景产生真实的折射与放大扭曲
- 玻璃表面覆盖一层可调的冷凝雾气，边缘柔和
- 文字框像贴在玻璃内侧，可自由排版、编辑、拖拽
- 一键导出带有完整文字合成的氛围画面

## 当前功能
- WebGL 实时雨滴折射：RGB 色差偏移 + LOD 分级（小/中/大水珠折射强度不同）
- Shader procedural 动态雨滴层：移植 rocksdanister/rain 的 `DropLayer2`，网格化密集雨滴零 draw call，与 CPU 物理雨滴互补
- Bokeh 径向失焦：大水珠边缘模拟放大镜虚化，雾气区域模拟景深 defocus
- Procedural 微水珠层：shader 内高密度静态微水珠，与 CPU 绘制的大/中水珠互补
- 雨夜湿润化后处理：降饱和、暗部蓝紫/亮部暖色 selective grading、底部地面反光、大气雨雾
- 闪电效果：随机闪烁 + 闪光爆发，支持开关
- 鼠标视差 + 镜头缩放：背景随鼠标轻微偏移，Zoom 缩放玻璃（长焦/广角）
- 可折叠玻璃态控制面板
- 图片/视频背景（cover / stretch 适配）
- 可拖拽文字层 + 导出 PNG

## 技术架构
- 渲染：p5.js WEBGL + 自定义 Fragment Shader
- 雨滴：CPU 物理 + GPU 法线贴图折射
- 雾气：预生成 Perlin 噪声纹理
- 文字：DOM overlay 实时编辑 + Canvas 合成导出
- UI：纯 CSS glassmorphism 控制面板
