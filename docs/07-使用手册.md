# 使用手册

## 快速开始

1. 克隆仓库后，在根目录启动本地服务器：
   ```bash
   python3 -m http.server 3000
   ```
2. 浏览器访问 `http://localhost:3000`。
3. 页面加载后自动显示程序生成的城市夜景背景 + 玻璃雨滴。

> **注意**：必须走 localhost 运行，直接打开 `file://` 会因 CORS 限制导致 shader 无法加载。

---

## 上传背景

- 点击右侧面板 **"上传背景图/视频"**，选择本地图片或视频。
- 支持格式：JPG、PNG、MP4、WebM 等常见图片/视频格式。
- 上传后背景立即替换，雨滴折射与雾气效果自动应用到新背景。
- 视频背景会自动循环播放，暂不支持导出为视频（仅导出当前帧为 PNG）。

---

## 调节参数

右侧面板提供以下实时调节项：

| 参数 | 范围 | 效果 |
|------|------|------|
| **雨滴密度** | 0 ~ 60 | 玻璃上静态水珠的密集程度 |
| **雨势大小** | 0 ~ 100 | 动态滑落雨滴的数量与速度（毛毛雨 → 暴雨） |
| **雨滴大小** | 4 ~ 25 | 动态雨滴的最大半径 |
| **折射强度** | 0 ~ 100 | 水珠对背景的放大扭曲程度 |
| **雾化程度** | 0 ~ 100 | 玻璃表面的朦胧雾气浓度 |
| **镜头缩放** | 80% ~ 150% | 模拟长焦/广角镜头对玻璃的透视变化（背景大小不变） |
| **背景模式** | Cover / Stretch | Cover 保持比例无黑边；Stretch 拉伸填满 |
| **闪电** | 开 / 关 | 随机闪电闪烁效果 |

所有滑块实时生效，无需点击确认。

## 文字排版

- **双击文字**进入编辑模式，可直接输入中文。
- **拖拽文字**移动位置，文字会吸附在玻璃上。
- **悬浮文字**时右上角出现删除按钮，点击移除。
- 点击面板 **"添加文字框"** 可新增副标题文本。
- 默认提供一组标题（HEAVY RAIN）+ 副标题，可直接修改内容。

---

## 导出图片

- 点击面板 **"导出 PNG"** 按钮，或按键盘 **`S`** 键。
- 导出时会自动合成 WebGL 画面 + DOM 文字层，生成一张完整图片。
- 文件名格式：`rainy-window-{时间戳}.png`。
- 大分辨率导出可能需要 1~2 秒，请等待提示出现。

## 交互操作

| 操作 | 效果 |
|------|------|
| 鼠标移动 | 背景产生轻微视差偏移，模拟透过玻璃观察街景 |
| 触摸屏滑动 | 同样触发视差（移动端适配） |
| 右下角 ⚙️ 按钮 | 收起/展开控制面板 |
| 面板右上角 × | 收起面板 |

## 性能与适配

- 项目自动监测帧率，若 FPS 持续低于 30，会自适应减少动态雨滴上限以保障流畅度。
- 低端移动设备建议将**雨势大小**和**雨滴密度**适当调低。
- 法线层与高光层已做隔帧绘制优化，shader 仍保持 60fps 流畅。

## 常见问题

**Q：打开页面后背景全黑，看不到效果？**
A：请确认通过 `localhost` 访问，而非直接双击打开 HTML 文件。若仍黑屏，请检查浏览器控制台是否有 shader 编译报错。

**Q：导出的图片没有文字？**
A：这是异步合成过程，请等待 1-2 秒。如果长时间无反应，请检查控制台是否有跨域或加载错误。

**Q：调节参数后画面变卡？**
A：将**雨势大小**和**雨滴密度**适当降低，或关闭闪电效果。低端设备建议雨势不超过 50、密度不超过 30。

**Q：水珠看起来像白色泡泡而不是透明透镜？**
A：请提高**折射强度**，并确保上传了有足够细节的背景图（纯色天空背景上折射效果较弱）。
